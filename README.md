# uid-tourism-website
# India Tourism Website 🇮🇳

A tourism website developed as part of a User Interface Design (UID) group project. The website helps users explore tourist destinations across different states and union territories of India.

## Features

* State-wise tourism information
* Tourist attraction pages with images
* Interactive navigation menu
* Sign In and Sign Up pages
* Google Maps integration
* User-friendly interface
